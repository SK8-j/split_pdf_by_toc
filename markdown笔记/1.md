# 知识点补充说明（按指令2要求）

1. **模型简化实例**：住房市场模型刻意忽略卧室数量等特征，仅保留位置差异这个核心变量。这种简化类似地图制作中的比例尺选择，1:1的地图虽精确但无用，经济学模型需要突出关键变量。

2. **均衡形成过程**：当市场价格为p*时，存在价格调整的自我纠正机制。若价格暂定为490美元（低于p*），会出现排队租房现象，此时房东有动力提价直至达到均衡点。这个调整过程可能持续数周或数月。

3. **需求曲线构建细节**：保留价格实证测量可通过问卷调查或历史交易数据。例如某学生最多愿付500美元/月，这包含其时间成本（通勤节省）和偏好价值（便利性效用）。

4. **税收短期效应**：50美元/年的房产税分析中，关键假设是供给完全无弹性（垂直供给曲线）。此时税收不会改变市场交易量，房东净收入下降50美元，形成"税收楔子"。

5. **帕累托改进案例**：在房租管制下，假设A先生（保留价300美元）与B女士（保留价500美元）通过400美元转租达成交易，这个改进不损害第三方利益，且使双方效用提升，证明原配置低效。

6. **垄断定价对比**：普通垄断者会限制产量至MR=MC点，如图1.7中p̂对应的数量，这必然小于竞争均衡量Q*。造成的无谓损失（Deadweight Loss）是效率损失的量化表现。
